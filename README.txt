Future TRS interactive presentation

Open index.html in a browser.

Navigation:
- Use the left-side vertical navigation.
- Use Previous / Next.
- Use left and right keyboard arrows.
- Click process flowchart stages to reveal details.
- Click personas to reveal assigned roles and least-privilege permissions.

Files:
- index.html: slide structure
- styles.css: TRA colour theme and animations
- app.js: transitions, navigation, process and persona interactions
- assets/tra-logo.png: TRA logo used for branding

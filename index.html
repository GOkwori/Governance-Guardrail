DCMG is the Production Release Gate

The implementer raises the change request only after UAT is complete and the business has signed off.

Three Independent Controls, One Joined-Up Governance Model

DCMG controls Production change, persona teams control least-privilege access, and the Data Governance Hub controls reference data ownership before data changes enter DCMG.

Change GovernanceAccess GovernanceData GovernanceSafer Production

Stakeholder Message

  • Business owns sign-off before Production.
  • DDaT implements only approved and tested changes.
  • Reference data changes are governed before release.
  • Power Platform access is persona-based and auditable.
